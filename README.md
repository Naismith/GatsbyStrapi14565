# Quick Start

## Starting Strapi

```
cd strapi
npm install
npm run develop
```

### You can access Strapi at http://localhost:1337

## Starting Gatsby

#### _Note_ Because Gatsby is using Strapi as a source, Strapi must have already started up.

```
cd gatsby
npm install
npm run develop
```

### You can access Gatsby at http://localhost:8000
